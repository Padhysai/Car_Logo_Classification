Heroku Deployment

Steps before Deployment

  • Add a file called 'gitignore' inside the project folder. This file contains the list of the files which we dont want to include in git repository.
.idea
  • Add a file called 'Procfile' inside the project folder this folder contiins command to run the flask application once deployed on the server.
web: gunicorn app:app
  • Open the command prompt and navigate to project folder. Enter "pip freeze > requirements.txt" this will generate the requirements.txt file.
pip freeze > requirements.txt
  • requirements.txt file helps the cloud app to install all the dependencies before starting the webserver.
