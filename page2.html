Project Overview

The main aim of this project is to classify the different brands by their logo. here we are using the Pretrained models provided by Keras. by using VGG16 model we will do the classification of logos.

Dataset info

I used images of 10 classes and each class consists of 1000 images for training and 200 images for testing with different augmentation techniques to make my model robust.

Names of the classes:

  • Buick
  • Chery
  • Citroen
  • Honda
  • Hyundai
  • Lexus
  • Mazda
  • Peugeot
  • Toyota
  • Volkswagen

Training Parameters

  • Training Data : 10,000 images
  • Testing Data : 2000 images
  • GPU : Nvidia P5000
  • Epochs : 50
  • Model : VGG 16
  • Training time : 150 min (approx)
